Rules

These rules govern how The People's Lottery operates. It's designed as a privacy-focused "referendum on surveillance," where every entry is a "vote" for private money. 50% of the pot goes to the winner, 40% rolls over to the next pot and 10% funds privacy tools (e.g., Zcash development or similar projects). All transactions are on Zcash shielded pools for anonymity‚Äîno KYC, no logs. We use open-source scripts for draws, verifiable by anyone via Zcash block hashes.

1. Overview

This is a donation-driven lottery creating awareness and funding privacy tech.

Users support private money and get entries by sending ZEC to the Pot Wallet address.

1 ZEC = 1,000 entries. More ZEC = more entries = better odds.

Pots start at 200 ZEC for the first draw, then doubles each time (400, 800, etc.).

If no winner (or unclaimed), the pot rolls over to the next draw.

Draws are triggered when the pot threshold is met.

Winner selected randomly using Zcash block hash for fairness.

2. Entries and Odds

Send any amount of ZEC (min 0.01 ZEC for 10 entries and max 10 ZEC for 10000 entries) to the Pot Wallet Address.

Use note format: "Country: [CODE] My Code: [YOURCODE] Friends Code: [FriendCODE]" to link entries to your shielded address and earn referral benefits.

Your Entry Code (generated from shielded address) ties entries to you. If you lose it you lose your claim.

Odds = (your entries / total entries). E.g., if pot is 200 ZEC (200,000 votes), 1 ZEC gives 0.5% odds for a 100x.

Extra entries from referrals: +25% if using a ref code; Super Perk Code owners get +50% (1.5x).

3. Caps per Entry Code

Max 10 ZEC per Entry Code per draw to keep it fair and prevent üêã domination.

Any ZEC sent over and above 10 ZEC is allocated entries into the next draw.

One code per shielded address; no multiple codes to bypass cap.

4. Draws and Payouts

First draw at 200 ZEC pot; next at 400 ZEC, then 800, etc.

Draw triggers when pot reaches each threshold (monitored via Zcash node).

Winner chosen by hashing the next Zcash block hash with total votes‚Äîprovably fair, no trust needed (open-source script on GitHub).

Payout: 50% of pot to winner's shielded address (linked to their code).

Rollover: 40% of pot rolls over to next draw to help grow the pot.

Do Good: 10% of every pot is paid to someone/team/initiative to help evolve the tools of privacy.

Rollover: Unclaimed pots add to the next threshold (e.g., if 200 ZEC unclaimed, next draw at 400 ZEC becomes 600 ZEC total).

5. Referrals

Share your code: Referrer gets 5% cashback (sent to their address) based on ZEC sent by code user.

Code user gets +25% extra votes (or +50% if the code used is a Super Perks Code).

No cap on referrals.
