Rules

These rules govern how The People's Lottery operates. It's designed as a privacy-focused "referendum on surveillance," where every entry is a "vote" for private money. 50% of the pot goes to the winner, 40% rolls over to the next pot and 10% funds privacy tools (e.g., Zcash development or similar projects). All transactions are on Zcash shielded pools for anonymity‚Äîno KYC, no logs. We use open-source scripts for draws, verifiable by anyone via Zcash block hashes.

1. Overview

This is a donation-driven lottery funding privacy tech.

Users support private money and get entries by sending ZEC to the purchase address.

1 ZEC = 1,000 entries. More ZEC = more entries = better odds.

Pots start at 200 ZEC for the first draw, then doubles each time (400, 800, etc.).

If no winner (or unclaimed), the pot rolls over to the next draw.

Draws are automatic when the pot threshold is met.

Winner selected randomly using Zcash block hash for fairness.

2. Entries and Odds

Send any amount of ZEC (min 0.01 ZEC for 10 entries and max 10 ZEC for 10000 entries) to the Pot Wallet Address.

Use note format: "Country: [CODE] My Code: [YOURCODE] Friends Code: [FriendCODE]" to link entries to your shieded address and get referral benefits.

Your Entry Code (generated from shielded address) ties entries to you. If you lose it you lose your claim.

Odds = (your entries / total entries). E.g., if pot is 200 ZEC (200,000 votes), 1 ZEC gives 0.5% odds.

Extra entries from referrals: +25% if using a ref code; Super Perk Code owners get +50% (1.5x).

Rollover: Unclaimed pots add to the next threshold (e.g., if 200 ZEC unclaimed, next draw at 400 ZEC becomes 600 ZEC total).
